# 목장월드 캐릭터 기준 시트 v1

예수님 NPC와 초보 목자 LV.1의 크기, 비율, 해상도, 스타일을 통일하기 위한 기준 자료입니다.

## 핵심 규칙

- 128x128 캐릭터 캔버스
- 실제 캐릭터 높이 약 104px
- 2.5등신 SD 치비 비율
- 3/4 front 기본 시점
- 같은 발 기준선
- 같은 타원형 발밑 그림자
- 따뜻한 진갈색 1~2px 외곽선
- 앱용 실제 에셋은 투명 PNG

## 사용법

Codex에게 이 ZIP과 `CODEX_PROMPT.md`를 함께 전달하세요.
기존 에셋을 당장 완벽히 바꾸기 어렵다면, 우선 홈화면 레이아웃과 캐릭터 표시 규칙부터 통일하세요.
