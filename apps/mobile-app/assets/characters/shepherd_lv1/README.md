# 목장월드 초보 목자 LV.1 에셋팩 v0

목장월드의 성장형 기본 목자 캐릭터 에셋입니다.

## 캐릭터 컨셉

- 이름: 초보 목자 LV.1
- 역할: 성장형 플레이어 캐릭터
- 톤: 따뜻함, 순수함, 초보 목자의 시작
- 스타일: SD 픽셀아트풍 수집형 모바일 RPG 캐릭터

## 파일 규칙

각 포즈는 4가지 크기로 export되어 있습니다.

- `*_512.png`: 마스터/상세 화면용
- `*_256.png`: 홈화면 캐릭터용
- `*_128.png`: 수집 카드/작은 UI용
- `*_96.png`: 아이콘/작은 슬롯용

## 주요 포즈

- `shepherd_lv1_idle_greeting`: 기본/인사
- `shepherd_lv1_pray`: 기도
- `shepherd_lv1_bless`: 축복
- `shepherd_lv1_teach`: 말씀/가르침
- `shepherd_lv1_walk`: 걷기
- `shepherd_lv1_think`: 생각
- `shepherd_lv1_joy`: 기쁨
- `shepherd_lv1_comfort_concern`: 걱정/위로

## 앱 내 권장 사용

홈화면 상태에 따라 아래처럼 매핑하세요.

- 기본 상태: idle
- 기도제목 확인: pray
- 목양 기록 완료: bless
- 가이드/튜토리얼: teach
- AI 분석 중/추천 중: think
- 목표 달성: joy
- 미연락 목원/주의 알림: concern 또는 comfort

## 주의

이 버전은 생성된 디자인 시트를 잘라 만든 1차 에셋입니다.
MVP 개발과 UI 연결 테스트용으로 적합합니다.
정식 릴리즈 전에는 각 포즈를 투명 배경으로 재생성하거나 디자이너가 픽셀 정리하는 것을 권장합니다.
