# Mokjang World Jesus NPC Assets

This folder contains normalized Jesus NPC pose assets for the mobile Mokjang World home screen.

- `*_256.png` is the default app render size.
- `*_128.png` is for thumbnails and compact UI.
- `*_512.png` is the master export.
- `manifest.json` maps app states to poses, moods, triggers, and speech messages.

Supported app states: `idle`, `bless`, `teach`, `guide`, `concern`, `joy`.
