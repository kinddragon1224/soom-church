# SOOM Graph Workspace Shift — 2026-03-26

## 왜 방향을 바꾸는가

기존에 진행한 구조 홈 / 교적형 대시보드는
기존 교적 시스템을 보기 좋게 정리한 수준에 머물 위험이 있다.

숨의 차별점은 단순 관리 화면이 아니라,
**옵시디언 그래프 모드처럼 관계와 군집이 보이는 교회 그래프 워크스페이스**여야 한다.

## 새 기준

숨은 다음처럼 정의한다.

> 교인, 가족, 목장, 교구, 부서/사역을 노드와 연결로 보여주고,
> 후속관리와 상태를 오버레이로 읽게 만드는 그래프형 워크스페이스

## 노드 종류
- 개인
- 가족
- 목장
- 교구
- 부서/사역

## 엣지 종류
- 가족 연결
- 목장 소속
- 교구 소속
- 사역 참여
- 후속관리 강조 연결

## 오버레이
- 새가족
- 정착중
- 심방 필요
- 후속 필요
- 안정
- 휴면

## 구현 원칙
1. 기존 카드형 구조 홈은 보조안으로 남긴다.
2. 메인 차별화는 그래프 뷰 프로토타입으로 검증한다.
3. 웹 우선 개발하되 모바일은 이후 별도 표현으로 분리한다.
4. 데이터 shape와 렌더링을 분리한다.

## 즉시 작업
1. 그래프 프로토타입용 데이터 shape 생성
2. 개인/가족/목장/교구/사역 노드 배치
3. 상태 오버레이 색상/강조 적용
4. 그래프에서 선택 시 상세 패널 연결
