  - 쇼츠 · 홍보영상
  - 유튜브 운영 세팅
  - 행사 랜딩 · 안내 제작

즉, 다음 세션에서 작업을 이어갈 때는 **최근 결정 우선**으로 보고,
공홈은 우선 서비스형 랜딩 완성도를 높이는 방향으로 진행한다.

## 3) 지금 우선적으로 해야 할 일

### 우선순위 높음
1. 모바일 히어로 간격/폰트 밀도 최종 조정
   - 모바일에서 문구가 아래로 너무 깔리지 않게 조정
   - 상하 패딩, 문구 간격, 버튼 간격 다듬기
   - 현재 `app/page.tsx` 기준 히어로 영역 계속 점검

2. 상품 카드에 가격 정보 추가
   - 고정가보다 `시작가(from)` 형태 권장
   - 예: `월 00만 원부터`, `00만 원부터`
   - 방문자가 대략적인 예산 감을 잡을 수 있게 하기

3. 문의 페이지 신뢰도 보강
   - 제출 후 어떻게 되는지 설명
   - 답변 예상 시간 안내
   - 문의 프로세스 2~3단계 짧게 표시
   - 사용자가 `작동 안 하는 데모 폼`처럼 느끼지 않게 하기

4. 홈 카피 압축
   - 2번째 섹션 제목 더 짧고 강하게 수정
   - `WHY SOOM` 설명 문장 2문장 정도로 압축
   - 모바일에서 문장 밀도 낮추기

### 우선순위 중간
5. 상품 페이지 비교표 더 결정형으로 압축
6. 버튼형 선택 UI의 선택 상태 표시 강화
