# MOBILE WORLD NPC PLAN (2026-04-14)

## 목적
목장월드의 정체성을 강화하되, 목양 실행 UX를 방해하지 않도록 NPC를 단계적으로 도입한다.

## NPC 1차 고정 범위
1. 예수님
2. 12사도 (가룟 유다는 배경 나무 근처 hidden 연출)
3. 여성 제자 일부 (막달라 마리아, 마르다, 마리아(베다니), 요안나, 수산나)

## 확장 범위 (2차 이후)
- 구약 인물 팩
- 창작 캐릭터 팩 (교회/목장별 커스터마이징)

## 연출 원칙
- NPC는 운영 정보 위에 올라타지 않는다 (월드 보조 레이어)
- NPC 연출이 명령창/실행 기록보다 앞서지 않는다
- 신학/공동체 정서를 해치지 않도록 상징적, 절제된 표현 유지

## 구현 원칙
- 카탈로그 파일: `apps/mobile-app/lib/world-npc-catalog.ts`
- 배치 계층: foreground / midground / background / hidden
- 단계 오픈: `unlockPhase`로 제어
