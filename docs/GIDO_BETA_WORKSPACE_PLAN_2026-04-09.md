# G.I.D.O BETA WORKSPACE PLAN - 2026-04-09

## 상태
이 문서는 G.I.D.O / 입력형 workspace V1 시도에 대한 아카이브다.
2026-04-09 기준으로 이 방향은 제품 기준 문서에서 내려갔다.

## 왜 내려갔는가
1. 목자가 직접 기입해야 하는 비중이 커졌다.
2. UI가 반복 노출이 많아 가독성과 집중도가 떨어졌다.
3. 기존 관리툴과 본질적으로 크게 다르지 않았다.
4. 숨의 목적 철학인 `인지 마찰 감소`, `대화 중심 운영`, `더 정확한 사람 관리`와 멀어졌다.
