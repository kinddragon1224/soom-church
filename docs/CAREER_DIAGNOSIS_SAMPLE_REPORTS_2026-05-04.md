# Career Diagnosis Sample Reports — 2026-05-04

목적: 무료 진단 이후 사용자가 “상세 리포트를 요청하면 무엇을 받는지” 즉시 상상할 수 있게 만드는 판매 증거물이다.

## 1. 학생/부모 샘플

가상 입력: 수학은 버겁지만 생명/환경 이슈에는 오래 몰입하는 고1. 부모는 의대/공대가 아니면 불안하다고 느끼는 상황.

한 줄 진단: 아이는 직업명보다 오래 붙잡을 문제 주제를 먼저 잡아야 하는 단계입니다.

핵심 막힘: 부모의 불안이 과목 선택을 빠르게 압박하면서, 아이가 버틸 환경과 탐색 순서가 분리되어 있습니다.

피해야 할 착각:
- 성적이 조금 나오는 과목만으로 진로를 고정하기
- 부모가 원하는 직업명을 먼저 정하고 활동을 끼워 맞추기

먼저 볼 방향:
- 생명·환경·지역 문제를 연결한 탐구 활동
- 데이터를 얹은 과학 글쓰기/발표
- 보건·환경·교육을 비교하는 진로 탐색

다음 7일 행동:
- 이번 주 아이가 오래 이야기하는 문제 3개 적기
- 관련 기사 2개를 읽고 질문 5개 만들기
- 부모가 금지할 말 2개와 물어볼 질문 2개 정하기

## 2. 20대/첫 커리어 샘플

가상 입력: 전공은 애매하고, 공모전·동아리·아르바이트 경험은 있지만 이력서에서 어떤 직무로 보여야 할지 막힌 26세.

한 줄 진단: 경험이 부족한 것이 아니라 경험을 시장 언어로 번역하는 기준이 부족한 상태입니다.

핵심 막힘: 활동을 많이 나열하지만 어떤 문제를 줄였고 어떤 결과를 만들었는지 증거 문장이 약합니다.

피해야 할 착각:
- 관심 직무를 매주 바꾸며 자소서만 다시 쓰기
- AI 활용을 툴 이름 나열로만 설명하기

먼저 볼 방향:
- 운영/마케팅 보조 직무에서 자료 정리 역량 증명
- 콘텐츠 기획 직무에서 사용자 반응 분석 증명
- AI를 활용한 반복 업무 개선 포트폴리오

다음 14일 행동:
- 경험 3개를 상황-문제-행동-결과로 다시 쓰기
- 반복 업무 하나를 AI로 30% 줄인 실험 기록하기
- 2주 안에 지원 직무 2개만 남기기

## 3. 40~50대/전환 샘플

가상 입력: 한 분야에서 18년 일했지만 산업 변화로 불안해진 48세. 새 자격증을 따야 할지, 기존 경력을 살려야 할지 판단이 안 서는 상황.

한 줄 진단: 새로 시작해야 하는 문제가 아니라, 이미 해온 일을 다른 시장의 언어로 다시 부르는 문제입니다.

핵심 막힘: 전환을 ‘처음부터 다시’로 이해하면서 기존 경력의 반복 역할과 문제 해결 자산까지 낮게 평가하고 있습니다.

피해야 할 착각:
- 불안해서 무작정 자격증부터 결제하기
- 과거 직함을 그대로 들고 다른 시장에 들어가기

먼저 볼 방향:
- 현장 경험을 교육/코칭형 서비스로 전환
- 운영 경험을 소규모 조직 관리 언어로 번역
- AI 도구를 붙여 문서화/매뉴얼화 자산 만들기

다음 7일 행동:
- 반복해서 해결한 문제 5개 적기
- 계속 가져갈 일/줄일 일/실험할 일 1개씩 고르기
- 재교육 후보를 결제 전 2주 실험으로 검증하기
