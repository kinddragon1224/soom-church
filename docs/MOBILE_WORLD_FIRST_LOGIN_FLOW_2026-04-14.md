# MOBILE WORLD FIRST LOGIN FLOW (2026-04-14)

## 왜 먼저 필요한가
가장 먼저 해결해야 할 핵심은 "목자가 로그인 후 무엇을 해야 하는지"를 화면에서 바로 안내하는 것이다.

## 1) 첫 접속 직후 (월드 생성 설정)
로그인 완료 -> `world-setup` 화면으로 이동

필수 입력:
- 교회 이름
- 지역
- 목장 이름
- 목원 목록 시작 방식(직접 입력 / CSV / 모라 명령)
- 초기 예상 인원

결과:
- 설정 저장 후 월드 진입
- 이후에는 바로 월드 홈으로 진입

## 2) 월드 생성 후 즉시 보여줄 액션
월드 홈 상단 아래에 "시작 액션" 카드 제공
- 이번주 출석 체크
- 긴급 기도/돌봄 정리
- 심방 일정 초안
- 목장 일정 기입

## 3) 목자가 접속해서 해야 할 것 + 제공받는 것
목자가 해야 할 것(입력/확정):
- 이번주 출석
- 긴급 기도/돌봄
- 심방 일정
- 목장 일정

AI가 제공할 것(자동 제안):
- 지난 주 데이터 기반 우선순위 브리프
- 후속/기도/심방 명령 초안
- 실행 결과 요약 및 다음 행동 제안
