# Career Diagnosis Offer Ladder — 2026-05-02

## Core strategy
Do not make 1:1 consulting the main product.

Use consulting as a high-priced anchor and scarce premium option. The scalable product is the asynchronous report: Sun-yong's judgment system delivered through Mora.

Positioning:

> 직접 상담은 제한적으로만 받습니다. 대신 더 많은 분들이 김선용의 판단 기준을 경험할 수 있도록 AI 커리어 해석 리포트로 먼저 제공합니다.

## Why this structure
- Protects Sun-yong's time.
- Prevents the brand from becoming “cheap AI test.”
- Makes the report look like accessible access to a premium judgment system.
- Keeps Mora and Sun-yong unified: Mora is not replacing Sun-yong; Mora delivers Sun-yong's interpretation system.
- Lets consulting remain expensive because it uses direct human time.

## Offer ladder

### 1. Free Diagnosis
Price: 0 KRW
Role: acquisition and segmentation
Promise: identify the current stuck pattern and one next action.
Delivery: instant web result
Sun-yong time: none

CTA:
> 내 상황 기준으로 보고 싶으면 무료 진단부터 해보세요.

### 2. Mini Report
Price hypothesis: 9,900–19,000 KRW
Role: first purchase / low-friction validation
Promise: one-line diagnosis + key blocker + next 3 actions.
Delivery: text or Notion-style report
Sun-yong time: minimal review or none after template stabilizes

Best for:
- early-career users with low budget
- people who are curious but not ready for a detailed report
- testing willingness to pay

Do not overpromise. This is not full career planning.

### 3. Detailed Report — Main Product
Price hypothesis: 39,000–59,000 KRW
Role: core scalable product
Promise: condition-based interpretation, 2–3 directions, mistaken paths to avoid, 7/14-day action plan.
Delivery: text/Notion first; PDF only after template stabilizes
Sun-yong time: short review, not a full consultation

Recommended default product name:
> 김선용 AI 커리어 해석 리포트

Track-specific names:
- Student/Parent: 우리 아이 진로 방향 해석 리포트
- Early Career: 내 경험을 직무 언어로 바꾸는 리포트
- Transition: 경력 재호명 리포트

### 4. Premium 1:1 Direction Session
Price hypothesis: 200,000 KRW+ initially; can rise later
Role: premium anchor, high-trust option, not the volume product
Promise: complex decision alignment with Sun-yong directly.
Delivery: limited slots, direct session
Sun-yong time: high

Rules:
- Do not discount casually.
- Do not make this the default CTA.
- Use only when the user has complex stakes or wants direct interpretation.
- Keep slots scarce.

Positioning:
> 직접 상담은 시간이 들어가는 프리미엄 옵션입니다. 먼저 리포트로 상황을 정리한 뒤, 복잡한 선택이 남을 때만 연결합니다.

## Pricing stance
Do not lock exact prices until we see behavior.
But use this ladder as the pricing architecture:
- Free: 0
- Mini: 9,900–19,000
- Detailed: 39,000–59,000
- Premium session: 200,000+

The strategic point is relative positioning:
- Consulting is expensive because it consumes Sun-yong's direct time.
- Reports are accessible because they deliver the judgment system without requiring a live session.
