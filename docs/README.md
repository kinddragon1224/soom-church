# Documentation Index

이 디렉터리는 PRD를 구현, 검증, 운영으로 연결하기 위한 실행 문서 세트다.

## 문서 역할

- `FEATURE_SPECS.md`: 기능별 범위, 사용자 흐름, 수용 기준
- `DOMAIN_MODEL.md`: 핵심 엔티티, 관계, 데이터 경계
- `API_CONTRACTS.md`: 현재 확정된 API 계약과 서비스 인터페이스
- `OPERATIONS_RUNBOOK.md`: 실행, 환경 변수, 점검, 장애 대응 기준
- `LAUNCH_CHECKLIST.md`: 런칭 전 확인 항목과 게이트
