# MOBILE WORLD NPC STYLE GUIDE 2026-04-15

## 목적
- 예수님 NPC를 **마스터 스타일**로 고정한다.
- 이후 NPC는 외형만 다르게, 스타일은 동일하게 제작한다.

## 마스터 기준
- 마스터 파일: `apps/mobile-app/assets/world-npcs/jesus-npc.png`
- 출력 캔버스: `80x120` (PNG, 투명 배경)
- 기본 포즈: 정면(Front-facing)

## 고정 규칙 (변경 금지)
1. 라인
- 외곽선 두께/샤프니스는 예수님과 동일
- 흐린 안티앨리어싱 금지

2. 명암
- 3단계 명암(밝음/중간/그림자) 유지
- 과한 그라데이션 금지

3. 픽셀 밀도
- 텍스처 디테일 과밀 금지
- 뭉개지는 리사이즈 금지 (최종 검수는 1x 실크기)

4. 색상
- 공통 중립 팔레트 유지
- 캐릭터 정체성 색은 포인트만 변경 (예: 머리두건, 띠)

5. 배경
- 완전 투명 필수
- 흰 박스/회색 매트/바닥판/그림자판 금지

## 캐릭터별 허용 변경 범위
- 얼굴 특징, 의상 색 포인트, 손 제스처, 표정
- 금지: 비율/라인 성격/렌더링 스타일 변화

## 검수 체크리스트
- [ ] 80x120 PNG 투명 배경
- [ ] 예수님과 같은 라인/샤프니스
- [ ] 정면 포즈 축 유지
- [ ] 확대 없이도 뭉개짐 없음
- [ ] 배경 잔여 픽셀 없음

## 운영 규칙
- 신규 NPC 제작 전: 마스터(예수님)와 1:1 비교
- 불일치 시: 스타일 우선, 캐릭터 디테일은 후순위
- 월드 반영 전: 월드 실제 스크린샷 기준 최종 확인
