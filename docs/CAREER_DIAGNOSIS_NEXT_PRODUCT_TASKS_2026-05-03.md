# Career Diagnosis — Next Product Tasks

_Date: 2026-05-03_

## Current product direction

The product should not become “Sun-yong sells cheap 1:1 consulting.”

Core structure:

> Free diagnosis → report product → premium 1:1 only when needed

Mora is not positioned as a separate AI replacement. Mora is the interface that delivers Sun-yong’s judgment system in a scalable form.

## Offer ladder

### 1. Free Diagnosis
- Role: entry point, segmentation, trust check
- Price: 0 KRW
- Output: stuck type + one next action
- CTA: “내 상황 기준으로 보고 싶으면 무료 진단부터 해보세요.”

### 2. Mini Report
- Role: first paid test
- Price hypothesis: 9,900–19,000 KRW
- Output:
  - one-line diagnosis
  - current blocker
  - next 3 actions
- Best use: low-friction purchase, especially for early-career users

### 3. Detailed Report
- Role: main scalable product
- Price hypothesis: 39,000–59,000 KRW
- Output:
  - current state one-line diagnosis
  - real blocker
  - 2 mistaken paths to avoid
  - 2–3 directions to consider
  - 7/14-day action plan
- This is the primary product to validate.

### 4. Premium 1:1 Direction Session
- Role: premium anchor, scarce option
- Price hypothesis: 200,000 KRW+
- Not the default CTA
- Used only when the case is complex or the customer wants direct interpretation

## Strategic reason

Consulting should be expensive because it consumes Sun-yong’s direct time.
Reports should be accessible because they deliver the same judgment framework asynchronously.

This protects time and makes the report feel like accessible access to a premium system, not a cheap AI test.

## Evaluation criteria for each sample

For each sample, judge:
1. Is the customer job urgent?
2. Is the output concrete?
3. Does it reduce decision risk, time, or anxiety?
4. Is Sun-yong’s trust visible without overlisting credentials?
5. Does it naturally lead to a paid report or premium session?

## Pricing rule

Do not decide price from intuition.
Price after reviewing sample quality and real behavior.

Possible outcomes:
- If output is light: mini report 9,900–19,000
- If output is decision-grade: detailed report 39,000–59,000
- If output requires real-time judgment: premium session 200,000+
