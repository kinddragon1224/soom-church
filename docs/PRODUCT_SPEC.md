# 숨 (Sum) 관리자 대시보드 MVP 명세

## 제품 정의
숨은 한국 교회의 교적·행정을 중심으로, 후속관리와 공동체 연결을 한 화면에서 돕는 **운영 허브**다.

## 철학
1. 첫 화면은 실무 대시보드
2. 관계 흐름/후속관리 중심
3. 콘텐츠는 모듈형 후순위

## 필수 화면
1. 로그인
2. 관리자 대시보드 홈
3. 교인 목록
4. 교인 상세
5. 교인 등록/수정
6. 교구/목장 관리
7. 신청 관리
8. 공지 관리
9. 활동 로그/리포트

## 핵심 엔티티
- users
- roles
- members
- households
- districts
- groups
- application_forms
- applications
- notices
- activity_logs

## members 필수 필드
- 이름, 성별, 생년월일, 연락처, 이메일, 주소
- 가족 연결(household)
- 교구(district), 목장(group)
- 등록일
- 직분
- 상태 태그
- 메모
- soft delete 여부

## 상태 태그 예시
- 등록대기
- 새가족
- 정착중
- 목장배정완료
- 봉사연결
- 휴면
- 심방필요

## 대시보드 구성
- 총 교인 수
- 이번 달 신규 등록 수
- 미처리 신청 수
- 후속관리 필요 교인 수
- 교구별 인원 분포
- 최근 등록 교인
- 최근 신청
- 최근 공지
- 빠른 작업 버튼

## 디자인 방향
- 데스크톱 우선 관리자 웹
- 좌측 사이드바 + 상단 헤더
- 카드/표/필터/상태배지 중심
- 단정하고 신뢰감 있는 톤
- ERP처럼 과도하게 딱딱하지 않게
