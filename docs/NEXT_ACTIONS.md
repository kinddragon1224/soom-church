## LATEST SESSION RECOVERY - 2026-04-11
### 현재 제품 정의
- 숨은 `목장 운영 개발 OS`다
- 기존 `gido`/기도 목장 베타는 실험 흔적이 너무 많아 과감히 폐기한다
- 새 베타는 기존 화면을 덧대지 않고, 기본 워크스페이스부터 새롭게 리뉴얼한다
- 핵심 구조는 `운영 입력 -> 구조화 -> 운영 상태판`이고, 이후 월드 씬은 높은 감성 퀄리티 기준으로 다시 설계한다

### 이번 세션 결정
1. `gido`는 더 이상 제품 기준이 아니다
2. `app/gido` 계열 베타는 은퇴시키고 새 beta 워크스페이스를 기본 진입으로 준비한다
3. 리빌드는 한 번에 많이 얹지 않고 단계별로 검증한다
4. 새 beta는 회사 운영 OS를 오마주하되, 도메인은 목장 운영 OS로 재구성한다

### 이번 세션 반영
- `gido` 전용 간편 로그인과 특수 post-login 진입을 제거함
- `gido` slug는 더 이상 옛 beta로 들어가지 않고 `/app/beta` 준비 화면으로 빠지게 바꿈
- `/app/beta` 최소 준비 화면 추가
- 운영 DB에서 `gido` 교회와 관련 테스트 운영 데이터, 멤버십, 전용 테스트 사용자 계정을 삭제함
- 과거 `gido` 기준 문서/스크립트 일부 제거 시작

### 바로 다음에 할 것
1. 코드베이스에 남은 `gido` 전용 파일/분기/문구를 계속 제거하기
2. 새 beta 운영 개발 OS의 기본 IA와 라우트를 세우기
3. 로그인/가입 후 기본 진입이 새 beta 워크스페이스로 열리게 연결하기
4. 이후 월드 씬은 별도 단계로 높은 퀄리티 기준에서 다시 시작하기

### 멈출 것
- 기존 `gido` 화면 위에 계속 덧대는 작업
- 낮은 퀄리티 임시 월드 실험을 메인 트랙으로 가져가는 것
- quick form, 관리 카드, 임시 보드 중심 사고
